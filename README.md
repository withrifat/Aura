## 📖 About Aura

Aura is a modern academic campus social platform designed to bring students, teachers, and campus communities together in one digital space.

Unlike traditional social media platforms, Aura focuses on academic and campus-related activities, making it easier for students to stay connected with their institution, access important information, engage with their community, and use productivity tools built specifically for academic life.

The goal of Aura is to create a centralized digital ecosystem where academic communication, social interaction, campus memories, and student resources can coexist seamlessly.

## 🛠️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/aura.git
```

### Install Frontend Dependencies

```bash
npm install
```

### Run Frontend

```bash
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```
